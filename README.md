# data-portfolio-social-analytics
Portafolio de ciencia de datos aplicada, con proyectos de análisis exploratorio, modelamiento predictivo y visualización en Python, Power BI y SPSS, orientados a problemas reales del ámbito social y educativo.

# Portafolio de Datos – Análisis Social y Educacional

Este repositorio presenta una selección de trabajos representativos en análisis de datos, ciencia de datos aplicada y evaluación basada en evidencia en contextos sociales y educativos.

Los proyectos aquí descritos corresponden a ejercicios metodológicos, demostrativos y versiones anonimizadas de trabajos reales, con fines exclusivamente profesionales.

## Contenidos

- Análisis estadístico descriptivo e inferencial
- Modelos predictivos en Python
- Visualizaciones y dashboards (Power BI)
- Resultados de análisis en SPSS
- Enfoque en educación, políticas públicas y programas sociales

## Tecnologías utilizadas

- Python (pandas, numpy, scikit-learn, matplotlib)
- Power BI
- SPSS
- Excel avanzado

## Nota sobre los datos

Los datos originales han sido anonimizados o simulados.
No se incluyen bases de datos institucionales reales ni información sensible.
